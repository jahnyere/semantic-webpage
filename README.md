# semantic-webpage

## Project Overview

This project demonstrates the use of semantic HTML elements to create a well structured webpage.
##Features
-structured webpage using semantic HTML elements
-clear separation of Header, Main content and Footer

## Semantic Tags Used
The following semantic tags were used in this project:
`<Header>`
`<Main>`
`<section>`
`<Footer>`

## What I Learned:
Through this project I practiced using:
- Structuring a webpage using semantic HTML
- Organizing content into meaningful sections
- Using headings and lists correctly
- Writingclean and readable HTML code
